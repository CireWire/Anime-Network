Welcome!

Anime Network is a sample site that utilizes AngularJS to create an Anime database. I'll be adding more directives and routes along the way to make it an SPA. I'm still learning to use AngularJS and I hope to soon incorporate it with Meteor, but until then I'm still sorting things out. I hope you can check it out and feel free to clone it for whatever reason you want!
